## Project Overview

This Power BI project analyzes E-Commerce sales performance and transforms raw data into interactive and actionable insights through dynamic dashboards and KPIs.

## Objectives

* Analyze sales and profit performance
* Identify top-performing products and regions
* Track shipping performance and customer segments
* Provide business insights for decision-making

## Dataset Challenges

One of the biggest challenges in this project was working with a large and complex dataset.

The project included:

* Data Cleaning & Transformation
* Handling missing and inconsistent values
* Building relationships between multiple sheets/tables
* Creating a proper data model for accurate analysis

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Excel

## Dashboard Insights

* Total Sales: 568K
* Total Profit: 285K
* Profit Margin: 47.72%
* Standard Class was the best shipping mode
* Office Supplies generated the highest profit

## Features

* Interactive filters and slicers
* Regional analysis
* Category and segment analysis
* Shipping mode analysis
* KPI tracking
* Map visualization
